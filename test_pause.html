Manual Testing Instructions

To test the pause/resume functionality:

  1. Click the loop button to activate it (should turn orange)
  2. Click play button - audio starts playing
  3. Click pause button (play button changes to pause when playing) - playhead should stop at current position
  4. Click play button again - should resume from pause position
  5. Let audio play to end with loop enabled - should restart automatically
  6. Click stop button - playhead should reset to start position and pauseTime reset to 0

All functionality has been implemented as specified.
